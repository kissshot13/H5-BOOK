##JS面向对象编程
 ###前言
 - 在JavaScript中是不区分类和实例的概念。而是通过原型（prototype）来实现面向对象编程。
 - 比如一个学生对象：
 ```js
 var Student = {
        name  : 'Stuent',
        height: 1.6,
        run   : function(){
            console.log(this.name + 'is running');
        }
    }
    
    var xiaoming = {
        name  : '小明'
    }
    
    xiaoming.__proto__ = Student;
 ```
 - 最后一行代码是吧xiaoming的原型志向了对象Student，看上去是从Student继承起来的一样。
